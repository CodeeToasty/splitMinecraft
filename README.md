# splitMinecraft
Little command line utility that given a length tells you all the way you can split it in equal parts using, for example, pillars.

usage: python3 splitMinecraft.py 69

output example:
  python3 pillar.py 69
  feasable solutions: 
  4 L####@####@####@####@####@####@####@####@####@####@####@####@####@####L
  6 L######@######@######@######@######@######@######@######@######@######L
  9 L#########@#########@#########@#########@#########@#########@#########L
  13 L#############@#############@#############@#############@#############L
  34 L##################################@##################################L
  69 L#####################################################################L
